# Calculadora IMC
Esse projeto foi desenvolvido em linguagem nativa do Android, o Kotlin. O projeto é um dos módulos dos curso do DevSpace onde chama-se 10D Challenge, nele é desenvolvido um aplicativo de cálculo de IMC.
